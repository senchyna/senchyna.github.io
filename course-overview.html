Course Overview

Week 1: Foundations in Deep Learning

Chapters 1 (Introduction), 2 (Linear Algebra), and 3 (Probability and Information Theory) from Deep Learning

Assignment: Answering conceptual questions and exercises to solidify understanding of foundational concepts

Week 2: Numerical Computation

Chapter 4 (Numerical Computation) from Deep Learning

Assignment: Introduction to Python through practical numerical computation exercises, applying basic algorithms and techniques

Week 3: Machine Learning Basics

Chapter 5 (Machine Learning Basics) from Deep Learning

Assignment: Build a basic machine learning model in Python, focusing on foundational ML principles

Week 4: Deep Networks: Basics

Chapters 6 (Deep Feedforward Networks) and 7 (Regularization for Deep Learning) from Deep Learning

Assignment: Implement and regularize a deep feedforward network using Python

Week 5: Advanced Deep Learning Techniques

Chapters 8 (Optimization for Training Deep Models) and 9 (Convolutional Networks) from Deep Learning

Assignment: Optimize and apply a convolutional neural network in Python

Week 6: Sequence Modeling and Practical Methodologies

Chapters 10 (Sequence Modeling: Recurrent and Recursive Nets) and 11 (Practical Methodology) from Deep Learning

Assignment: Develop and apply an RNN in Python for a sequence modeling task
